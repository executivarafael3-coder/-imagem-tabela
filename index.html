Visualizador da Imagem

Abra este arquivo em qualquer navegador. A imagem está armazenada localmente na mesma pasta.

Dicas de uso:
  • Para abrir localmente, dê duplo clique em index.html ou inicie um servidor simples: python -m http.server.
  • Para compartilhar entre computadores, compacte a pasta (já incluída aqui) e envie por e-mail, Google Drive, Dropbox ou hospede no GitHub Pages / Netlify.
